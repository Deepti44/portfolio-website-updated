📌 Project Overview

This project is a responsive portfolio website developed using HTML, CSS, and JavaScript.

Initially built with HTML and CSS, the website was enhanced with JavaScript to add dynamic interactivity including hobby management, form alerts, and dynamic footer date-time display.

The project is managed using Git and GitHub with three branches:

dev → Development branch

stage → Testing branch

main → Final production branch

Workflow followed:

dev → stage → main

🎯 Assignment Objective

Enhance the existing portfolio website by adding JavaScript interactivity.

🛠 Technologies Used

HTML5

CSS3

JavaScript (DOM Manipulation & Event Handling)

Git & GitHub (Version Control)
